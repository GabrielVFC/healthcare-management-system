### üöÄ Key Technical Features

* **Comprehensive Schema Design (DDL)**: Implemented structured tables using Primary Keys, Foreign Keys, and Autoincrement constraints to ensure data integrity.
* **Advanced Data Querying (DQL)**: Developed complex queries using multiple `JOIN` operations (Inner, Left) to connect patient records with medical history.
* **Analytical Insights**: Integrated aggregate functions (`AVG`, `COUNT`, `MIN/MAX`) and date manipulation using `JULIANDAY()` to calculate metrics like average patient age.
* **Business Logic**: Created advanced filtering using `LIKE`, `IN`, `BETWEEN`, and `GROUP BY/HAVING` clauses to extract actionable medical data.
